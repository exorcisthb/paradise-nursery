# 🌱 Paradise Nursery Shopping App

This is a React-based shopping cart application for an online plant shop called Paradise Nursery.

## Features
- Browse plants by category
- View plant details (name, image, price)
- Add plants to cart
- Update quantity
- Remove items from cart
- Calculate total price

## Technologies Used
- React
- Redux Toolkit
- CSS
